# Projects from [my YouTube channel](https://www.youtube.com/@BlockerLockerYT)

* [**Data Pack Directory**](datapacks/!directory.md)
* [**Resource Pack Directory**](resourcepacks/!directory.md)

This repository is to store some of the commands, data packs, and resource packs from my YouTube videos. Anything in this repository will likely NOT get updates or continued support, and have the version they were made for stated in their respective directories (listed above). Most projects will likely work in most future versions of the game, but may not support new blocks/items/mobs that have been added unless the packs are modified, which I cannot commit to doing in the long-term. If you have any other issues with these projects, or just have questions, feel free to join my Community Discord: [https://discord.gg/EBEBtBVKCK](https://discord.gg/EBEBtBVKCK)
